# VisionNodes Studio Website

This is the static website for VisionNodes Studio, designed to be hosted on GitHub Pages.

## Tech Stack
- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Development
1. `cd website`
2. `npm install`
3. `npm run dev`

## Deployment
Run `./deploy.sh` to build the production bundle. The contents of the `dist` folder should be pushed to the `gh-pages` branch.
